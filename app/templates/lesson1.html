Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is a coding language that is used to create webpages.
In fact, it was used to create this tutorial! Here is an example of some basic html. Click "See Result" to see how it looks on a webpage: 


<!DOCTYPE html>
<html>
<head>
   <style>
       h1 {
           background: yellow;
       }
   </style>
</head>
<body>
   <h1>My Webpage</h1>
   <br>
   <p style="color:blue">Here is some blue text.</p>
</body>
</html>




In this tutorial you will learn how to create webpages with HTML! Have Fun!
