Interactive HTML

Links and Buttons

Links

The <a> tag is used to create links. It has a attribute called href, which is the url of the page that it links to. Inside the content of the link, you write the text that the browser will display. For example, <a href="https://google.com">Link To Google</a> makes this: Link To Google

The link also has a target attribute. This specifies where you want the page to open. If you want the link to open in a new tab, you must set target="_blank". Here's an example: <a href="https://google.com" target="_blank">Link to Google</a> does this: Link to Google


Button

Next, lets talk about the <button> tag, which is exactly what it sounds like: a button! The button tag has a type attribute, which specifies what kind of button it is.
The default attribute value is button. This is a list of type attributes:

  • type="button". This is the default value and it does not have any special properties.
  • type="submit". A submit button can be put inside a form, and when it is clicked on, the form is submitted. Read more about forms here.
  • type="reset". A reset button can also be put in a form to reset all its input values. Learn more here

The button element also has an onclick attribute. This contains some JavaScript to execute when the button is clicked. You can learn JavaScript here.
