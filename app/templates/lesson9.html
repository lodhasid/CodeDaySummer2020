HTML Tables and Lists

<table>, <ol>, and <ul>

Storing data in your webpages is very useful and there are two easy ways to do it:

HTML Tables:

HTML tables are a way to make a chart with columns and rows in HTML. The table must go inside a <table> tag. Inside of the <table>, there should be <tr> tags. That is a table row.

Inside the table row there are two kinds of tags you can put: <th> and <td>. The first one is a table header, and it usually goes in the first row. The second one is table data, and it used for all the rest of the data in the table.

Note: You do not need to have <th> tags in the table.

Here's an example of a table: 


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>

<table width="500" height="250">
    <tr>
        <th>First Name</th>
        <th>Last Name</th>
        <th>Favorite Color</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>Doe</td>
        <td>Green</td>
        <td>37</td>
    </tr>
        <tr>
        <td>Bob</td>
        <td>Bobby</td>
        <td>Black</td>
        <td>26</td>
    </tr>
        <tr>
        <td>Sam</td>
        <td>??</td>
        <td>Green</td>
        <td>42</td>
    </tr>
</table>

</body>
</html>
