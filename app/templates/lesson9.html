HTML Tables and Lists

Storing data in your webpages are useful but they aren't useful if you don't know how to do it.
Making a table is a bit complex but pretty easy at the same time.
you start off with the <table> tag and there are a few more tags you have to put in the table.
each <tr> tag is a table row.
<th> and <td> tags always go in <tr> tags. <tr> tags always go in <table> tags.
the <td> tag makes text in it regular and left aligned.
the <th> tag makes text in it bold and centered.
The <list> tag is much easier than the <table> tag.
You just put a <ol> tag or <ul> tag.
the <ol> tag makes an ordered list and the <ul> tag makes an unordered list.
inside one of the the list tags, you put a <li> tag. this is the tag that creates each row of your list.
whatever you write in here will be part of one row of your list. You can create as many <li> tags as you want!
(in the ol or ul tag of course!)
