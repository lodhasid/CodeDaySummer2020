CSS Backgrounds

Background Colors and Images

Background Colors

Creating background colors in CSS is very simple. The property is just:
background-color: color;
Here's an example: 


<!DOCTYPE html>
<html>

<head>
<title>CSS Example</title>
<style>
    body {
        background-color: #448844;
    }
    
    h1 {
        background-color: lightyellow;
        padding: 7px
    }
</style>
</head>

<body>
<h1>This is a header</h1>
</body>

</html>

Background Images

Creating background images is also very easy. The CSS property is:
background-image: url("Image URL"); 


<!DOCTYPE html>
<html>

<head>
<title>CSS Example</title>
<style>
    body {
        background-image: url("/static/bgImage.jpg");
    }
    
    h1 {
        background-color: lightyellow;
        padding: 7px
    }
</style>
</head>

<body>
<h1>This is a header</h1>
</body>

</html>

However, this makes the image keep its original size. It went past the bottom of the page, and it did not go all the way to the right. This causes the background image to repeat in the unfilled space.

We can set a CSS property called background-size to fix this.

There are three different ways to do this.

"background-size: 100vw 100vh", which sets the width and height to the screen width and height. This makes the image stretch/shrink to fit the screen: 


<!DOCTYPE html>
<html>

<head>
<title>CSS Example</title>
<style>
    body {
        background-image: url("/static/bgImage.jpg");
        background-size: 100vw 100vh;
    }
    
    h1 {
        background-color: lightyellow;
        padding: 7px
    }
</style>
</head>

<body>
<h1>This is a header</h1>
</body>

</html>

background-size: contain, which causes the background to shrink to fit in the page, but still repeats some parts. To make it not repeat, put background-repeat: no-repeat: 


<!DOCTYPE html>
<html>

<head>
<title>CSS Example</title>
<style>
    body {
        background-image: url("/static/bgImage.jpg");
        background-size: contain;
    }
    
    h1, p {
        background-color: lightyellow;
        padding: 7px
    }
</style>
</head>

<body>
<h1>This is a header</h1>
<p>Try adding "background-repeat: no-repeat;" to the body's CSS</p>
</body>

</html>

background-size: cover which shrinks/expands image to cover the full page, cutting out some parts:


<!DOCTYPE html>
<html>

<head>
<title>CSS Example</title>
<style>
    body {
        background-image: url("/static/bgImage.jpg");
        background-size: cover;
    }
    
    h1 {
        background-color: lightyellow;
        padding: 7px
    }
</style>
</head>

<body>
<h1>This is a header</h1>
</body>

</html>
