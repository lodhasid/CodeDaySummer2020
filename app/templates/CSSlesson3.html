CSS Selectors

What are CSS Selectors?

A CSS selector is some text that can select specific HTML elements from a document. Here are some things you can select with CSS selectors:

  • Elements with specified tag name: "tagName"
  • Element with specified id: "#id"
  • Elements with specified class: ".class"
  • Element inside specified element: "parentSelector childrenSelector"
  • Element with specified attribute: "[attr="value"]"
  • Element in specified element state: ":state"
  • Everything: "*"

So the CSS selector "#header" would select the element with id header. And ".myClass p" would select all <p> tags inside an element with class myClass. An example of using these in actual CSS would look like this: 


<!DOCTYPE html>
<html>

<head>
    <title>CSS Example</title>
    <style>
        #header {
            color: blue;
        }
        
        .myClass p {
            font-size: 30px;
        }
    </style>
</head>

<body>
<p id="header">This has id header so it is blue</p>
<h1>This does not have id header</h1><br>

<div class="myClass">
    <p>This is a paragraph inside an element with class myClass</p>
    <h5>And this is not a paragraph</h5>
</div>

<p>And this is not inside an element with class myClass</p>

</body>
</html>

You can also combine multiple selectors, for example: "h1.header" would select only elements that were <h1> tags and had class "header", and "div p[title="paragraph"]" would select all paragraphs with the attribute title="paragraph" that are inside a <div> element. Here's an example: 


<!DOCTYPE html>
<html>

<head>
    <title>CSS Example</title>
    <style>
        h1.header {
            padding: 20px;
            background-color: lightblue;
        }
        
        div p[title="paragraph"] {
            font-family: impact;
        }
    </style>
</head>

<body>
<p class="header">This has class header but is not an h1 tag</p>
<h1>This doesn't have class header</h1>
<h1 class="header">This is an h1 with class="header"</h1><br>

<p title="paragraph">This is not in a div</p>
<div>
    <p title="paragraph">This is a p tag with title paragraph that is inside a div</p>
    <p>This doesn't have title="paragraph" so it will not be styled</p>
</div>
</body>
</html>
