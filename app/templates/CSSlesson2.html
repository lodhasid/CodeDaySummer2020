CSS Syntax

Selectors, Properties, and Values

Internal and External CSS (inline CSS will not be covered in this tutorial) have this syntax: 


selector {
    property: value;
    property2: value2;
    property3: value3;
    ...
}

selector2 {
    property: value;
    property2: value2;
    property3: value3;
    ...
} ...

Selectors are a way to select which elements that the style applies to. If you want to apply styles to multiple different selectors, seperate the selectors with commas.

Properties are the names of the styles, like "color", or "font-size".

Values are the value you want to set to each property; in "font-size: 30px;", the value is "30px". Together, a property and a value make a style rule.
