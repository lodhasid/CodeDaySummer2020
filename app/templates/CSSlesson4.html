CSS text styling

Fonts and Colors

To style text, there are many CSS properties you can use:

  • font: size font;
  • color: color;
  • font-weight: bold, light;
  • text-decoration: type decorations;
  • -webkit-text-stroke: size color;
  • font-variant: small-caps, all-small-caps, or none;
  • text-shadow: x-offset y-offset blur-size color;
  • text-align: center, left, or right;

The font property changes the font of some text. An example of its value could be "20px Arial".
Note: If the font name has spaces in it, like Comic Sans MS, then you must put it in quotes.

The color property changes the color of text. Its value can be in these forms: rgb(), rgba(), hsl(), hsla(), color name, or hex format (see CSS colors). For example, its value could be "rgb(0, 179, 7)".

The font-weight property can change the boldness of text; its values can be either "bold" or "light".

The text-decoration can make an underline, overline, or line through the text. And it can be normal, dotted, double, dashed or wavy. If you wanted to make a double underline, you would just write "text-decoration: double underline" . And if you wanted to make a dotted underline AND overline, then you would put "text-decoration: dotted underline overline".

-webkit-text-stroke makes a border of the text, in the form "size color".
For example "-webkit-text-stroke: 2px red" would make a 2 pixel wide red border around the text.

The font-variant property makes text have small caps. The "small-caps" value makes all the lowercase letters small caps the uppercase letters regular caps. And the "all-small-caps" value makes ALL letters small caps. The value of "none" makes all the letters normal.

"text-shadow" makes a shadow of the text. It is written like this: "x-offset y-offset blur-size color". So "5px 5px 3px blue" would make this a blue shadow 5 pixels right and down from the text, blurred slightly.

And finally, text-align. This aligns the text to either

text-align: left

text-align: center

or text-align: right

Here's an example of some styled text:


<!DOCTYPE html>
<html>
<body>

<head>
<style>
    h1#styled {
        font: 50px Arial;
        font-weight: bold;
        -webkit-text-stroke: 2px yellow;
        text-shadow: 7px 7px 8px rgb(90, 90, 90);
    }
</style>
<head>

<body>
    <h1 id="styled">Styled Text</h1>
</body>
</html>
