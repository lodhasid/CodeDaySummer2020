Styling HTML Elements

Did you know that you can style HTML elements to make them look better? Just look at the heading on the top of the page and imagine if it looked like this:

HTML Styles

Looks terrible, right? Well, to make elements look better, you just have to add a style attribute to the element and set it equal to a list of style property-value pairs. It looks like this:

style="property: value; property2: value2; property3: value 3..."

The color: color property changes the color of text.

The background-color: color property changes the color of the background of an element.

The font-family: font property specifies the font of an element.

The font-size: #units property changes the size of an element's text.

The text-align: left | center | right property aligns the text of an element to the left, center, or right.

The border: #units type color property creates a border around an element.

The padding: #units property adds some space around an element, but inside the border.

The margin: #units property adds some space around an element, outside the border.

#units means a number and a unit of measurement. The unit of measurement can be px (pixels), in (inches), mm (millimeters), cm (centimeters), % (percentage of parent width/height), vw (percentage of screen width), vh (percentage of screen height). For example, you could have font-size: 30mm which would make the font size 30 millimeters.

The border type can be solid, dashed, dotted, ridge, or groove. These are some examples of a few style properties:

  • color: red
  • background-color: lightgreen
  • font-family: impact
  • font-size: 25px
  • text-align: center
  • border: 2px solid black

Or with multiple style properties:

  • background-color: lightgreen; padding: 10px
  • color: blue; font-family: impact
  • padding: 20px; border: 5px ridge yellow; color: white; background-color: blue

These are just a few styles. There's a lot more properties and combinations of styles you can make!
