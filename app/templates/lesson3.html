HTML Attributes

What is an attribute?

An HTML attribute gives additional information about an element. An element's attributes go in between the tag name and the > sign:
<tagName attribute="value">content</tagName>

If an element has multiple attributes, then they are separated by spaces.

An example of an attribute is the title attribute. Almost all elements visible on the page have a title attribute. When you hover over an element, it displays the text in the title attribute as a tooltip. Here's an example: 


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>

<p title="This is in the title attribute">Hover over me!</p>

</body>
</html>

Another example of attributes are the class and id attributes. Classes are used to classify elements with a class name, and the id attribute gives an element a unique id. They are useful for JavaScript and CSS.
