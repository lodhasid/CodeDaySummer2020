Basic HTML

HTML Tags

Things you write in HTML are called tags. Almost all HTML tags look like this:

<tagName attributes>content</tagName>

Or if there is no content:
<tagName attributes />

TagName is the name of the HTML element. <tagName attributes> is called the opening tag, and </tagName> is the closing tag. Anything in between is the content. Content can just be text, or it can be element(s). HTML Attributes will be covered in the next lesson.

This lesson will be about basic HTML tags:


HTML Documents

An html document is made with a .html file:

Every HTML document starts with a Document Type declaration: <!DOCTYPE html>. This is NOT A TAG. You do not need to have </!DOCTYPE html> at the end of your file.

After that, the entire document must be inside a <html> tag.

There must also be a <head> tag inside the <html> tag. This contains additional data about the document. Inside the <head> tag, there should be a <title> tag. This is the title of your webpage, and is displayed at the top of the browser tab:

Inside the <html> tag, usually after the <head>, there must be a <body>. This contains all visible content. Anything outside the <body> tag will NOT be visible.


HTML Headers and Paragraphs

The header tag makes headings and looks like this: <h#>. # is a number that is the size of the header. For example, <h1> makes the biggest heading:

This is Header 1


<h2> makes a smaller heading.
<h3> makes a even smaller heading.
<h6> is the smallest:

This is Header 6

The <p> tag makes a paragraph that you can write lots of text in. The <p> tag was used to write this very text!

You can use the <br> tag to create a newline. This element is empty--it has no content--so it does not need a closing tag.

Example of HTML tags:


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>
    <h1>This is Header 1</h1>
    <h2>This is Header 2</h2>
    <h3>This is Header 3</h3>
    <h4>This is Header 4</h4>
    <h5>This is Header 5</h5>
    <h6>This is Header 6</h6>
    <p>This is a paragraph<br>and this is a newline</p>
</body>
</html>
