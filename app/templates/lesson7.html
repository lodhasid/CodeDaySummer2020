Inline vs Block Elements

What are They?

Inline vs Block:

An inline element is an element that can go inside any other element. It does not start from a new line, and it shrinks to the size of its content. However, block elements are the opposite: they cannot go inside inline elements, and they always start on a newline. They also take up as much width as is available to them. An example of this is headers (<h1> - <h6>). Headers always start on a newline, and they take up the entire width of the page. An example of an inline element is the link (<a>) element. It can go inside any element, and it does not take up any more space than it needs to.

Divs and Spans

The <div> tag is a block element often used for grouping different elements, and is often used for applying styles to them or to itself. Really, it is just a block element that can have any content inside it.

<span> tags are inline and are usually used to select part of the text content of an element and do something to it, such as applying a style to it.

Here is an example:


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>

<div style="background-color: lightblue;padding: 10px">
    <h1>This is a header</h1>
    <p>This is a paragraph.</p>
</div><br>

<p>Here is some text in a span element: <span style="color: blue">Blue text blue text blue text...</span></p>

</body>
</html>
