What is CSS?

Cascading Style Sheets

CSS stands for Cascading Style Sheets. It is a way to style HTML elements. There are 3 ways to write CSS: inline, internal, and external. This lesson will go over these 3 ways.

Inline CSS

This type of CSS was briefly explained in our HTML Tutorial. It is written as an HTML attribute called style. Inside the style attribute, you put your CSS. For example:


<p style="Your CSS Here">Paragraph</p>

Internal CSS

This type of CSS is more common and is easier to use. For this type of CSS, you need to put a <style> tag inside the <head> tag. Your CSS goes inside the style tag. It would look something like this:


<head>
<style>
    Your CSS Here
</style>
</head>

External CSS

External CSS uses a .css file with your css in it. And you need a <link> tag inside the <head> tag of your html document. The link tag is empty and must have attributes rel="stylesheet" and href="URL". URL is the URL to your .css file. Here's how it would look:

myCSS.css: 


Your CSS Here

myWebpage.html: 


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
    <link rel="stylesheet" href="myCSS.css">
</head>
<body>
    <p>This could be styled in myCSS.css</p>
</body>
</html>

There are advantages and disadvantages to each of these ways, which you can see here. In this tutorial, all examples will be written using internal CSS.
