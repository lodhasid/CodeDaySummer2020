CSS Padding, Border, and Margins

What are they?

Borders

An element's border can be specified with CSS. You can specify each side of an element's border with these CSS properties:

  • border-left
  • border-right
  • border-top
  • border-bottom

Or, if you want them all to be the same, you can just use the border property.

The values for each of these properties is in the form "units type color". For example, "border: 2px dashed green" would create a green dashed border 2 pixels wide, and "border: 3mm solid blue" would create a solid blue border 3 millimeters wide.

Padding

Padding is space around the content of an element. Padding is inside the element's border. The padding property's value is like this: "up right down left" where each one of those is the padding, in any units, on that side of the element. If the padding is the same on all sides, then it can just be written like this: "padding: units", where units is the padding on all sides.

For example, "padding: 5px" would make five pixels of space between an element's content and its border. "padding: 5mm 2mm 5mm 2mm" would make 5 millimeters of space on the top and bottom, and 2 millimeters of space on the left and right.

Margins

Margin is the space around an element, outside of its border. The margin property's value is written the same way as the padding property's.
Note: The background or size of an element will not include the margin.
