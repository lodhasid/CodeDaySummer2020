HTML media

Images, Videos, and Audio

Images

The <img> tag creates an image. It has no content, so it can be written without a closing tag.

The <img>'s most important attribute is src. This specifies the source url for the image.

Another attribute of an image element is alt. If the browser is, for whatever reason, unable to load the image, it will display the text inside the alt attribute.

Lastly, there is the width and height attributes. They contain a number (in pixels) of the width or height of your image. Note: if you only specify one of them, the aspect ratio of your image will be preserved.

Here's an example of an <img> element: 


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>
<img src="/static/image.png" alt="Test Image" width="150">
<img src="/static/doesNotExist.png" alt="Test Image" width="150">
</html>
</body>

Video & Audio

Next lets talk about the <video> tag. The video tag makes a video that users of your webpage can watch. Inside the video tag, you must put one or more <source> tags. Each <source> tag is an empty element with two important attributes. Src, which, just like the images, is the source of the video, and type, which specifies the video format.

If your video is a .mp4, then the <source> must have type="video/mp4". If it is a .ogg, then you must write type="video/ogg". Same thing for other video formats. Note: if you have multiple <source> tags, the browser will use the first one it finds.

The audio tag is almost exactly the same as the video tag, but the <video> becomes <audio>, and the type attribute of the source becomes type="audio/fileFormat", instead of type="video/fileFormat". This will create an audio player on the page that plays the audio in the <source> tag.

Note: File formats like .m4a or .mp3 are just written as type="mpeg"

Both <video> and <audio> have two important attributes: controls, and autoplay. These attributes are not regular attr="value" attributes. You just need to write the attribute name without the '="value"' part, because they do not have a value.

If you include the controls attribute, the audio/video will have a play button, pause button, etc. If you include the autoplay attribute, it will play as soon it loads.

Examples of video and audio: 


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>
<video width="300" controls>
    <source src="/static/video.mp4" type="video/mp4">
</video>
<audio width="300" controls>
    <source src="/static/audio.m4a" type="audio/mpeg">
</audio>
</body>
</html>
